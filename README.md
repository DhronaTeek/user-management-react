User Management App

A simple React application that performs CRUD operations (Create, Read, Update, Delete) on users using JSON Server as a mock backend.

Features

View all users

Add new user

Edit existing user

Delete user

Basic error handling

Technologies Used

React

Axios

JSON Server

CSS

How to Run
1. Install dependencies
npm install
2. Start JSON Server
json-server --watch db.json --port 8080
3. Start React App
npm start

The app will run at:

http://localhost:3000

API runs at:

http://localhost:8080/api/users
