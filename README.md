# User Management App

A simple React application that performs CRUD (Create, Read, Update, Delete) operations on users using JSON Server as a mock backend.

## Tech Stack

- React
- Axios
- JSON Server
- CSS

## Features

- View all users
- Add new user
- Edit user details
- Delete user
- Basic error handling

## Setup Instructions

1. Install dependencies

npm install

2. Start JSON Server

json-server --watch db.json --port 8080

3. Start React App

npm start

Frontend runs on:
http://localhost:3000

Backend API runs on:
http://localhost:8080/api/users
